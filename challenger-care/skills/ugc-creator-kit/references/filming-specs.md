# Filming Specs

Standard technical requirements for all UGC video briefs. Copy the relevant spec table into Section 10 of the creator brief, adjusting platform and length as needed.

---

## Standard Spec Table (Instagram Reels / TikTok / Facebook Reels)

| Spec | Requirement |
|------|-------------|
| Orientation | Vertical 9:16 (portrait — fill the full frame, no black bars) |
| Resolution | 1080p minimum / 4K preferred if your phone supports it |
| Frame rate | 30fps or 60fps |
| Lighting | Natural light preferred (near a window) / Ring light acceptable / Avoid harsh overhead shadows on face |
| Audio | Speak clearly and at a natural pace — no background music while talking (we add music in post-editing). Reduce background noise where possible. |
| File format | .mp4 — raw and unedited |
| Captions | None — do not burn in subtitles (we handle captioning in editing) |
| Filters | None — deliver clean, unfiltered footage |
| Music | None — do not add music to the raw file |
| File naming | [Brand] \| [Creator Name] \| Concept[N] \| [YYYY-MM-DD] |
| Delivery method | [AM to fill in: Google Drive / WeTransfer / Dropbox — include link] |
| Footage deadline | [AM to fill in date] |

## Platform-Specific Length Guide

| Platform | Recommended length | Notes |
|----------|--------------------|-------|
| Instagram Reels | 30–60 seconds | Aim for 60s for educational content, 30s for punchy hooks |
| TikTok | 30–60 seconds | Shorter performs better unless strong narrative |
| Facebook Reels | 30–60 seconds | Same as Instagram |
| YouTube Shorts | 30–60 seconds | Can go up to 60s; keep it tight |
| Story / Square format | 15–30 seconds | If brief requests story format, shoot vertical 9:16 |

## B-Roll Guidance

B-roll is supplementary footage that supports the voiceover or main talking-head clip. Unless the brief specifies otherwise, deliver B-roll as separate clips (not edited together with the main take).

Common B-roll types:
- **Product in hand** — close-up of the product being held, unboxed, or inspected
- **Product in use** — hands using it, wearing it, interacting with it in a real setting
- **Setting shots** — desk setup, coffee shop table, home environment
- **Screen/device shots** — laptop screen, phone in hand, reading materials
- **Reaction/expression** — natural facial expressions of comfort, focus, relief

Deliver all B-roll clips raw, unnamed except for: `[Brand] | [Creator Name] | B-Roll | [YYYY-MM-DD]`

## Common Quality Issues to Avoid

| Issue | How to avoid |
|-------|-------------|
| Shaky footage | Use a phone stand, tripod, or prop phone against a stable surface |
| Poor lighting | Film near a window, face the light source — never have it behind you |
| Echo / reverb | Film in a room with soft furnishings (bed, sofa, curtains absorb sound) |
| Wind noise (outdoor) | Avoid filming outside on windy days; use a windscreen if available |
| Frame too wide | Fill the frame — face and upper body only; don't stand too far back |
| Looking off-camera | Look directly into the phone camera lens, not at your screen |
