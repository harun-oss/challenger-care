# Creative Testing Workflow — Canva, Specs, and Monthly Cadence

## Monthly Experiment Workflow

### Week 1: Plan + Brief
- Define experiment number, type, and hypothesis
- Pull control elements from main campaign (best headline + best primary text)
- Define new elements to test (4-5 images/videos + 2 new headlines + 2 new primary texts)
- Build Ad Creative Overview deck — send to client for approval

### Week 2: Build + Launch
- Build assets in Canva Pro (account)
- Upload to Ads Manager as new DCT ad set
- Confirm permutation count ≤ 20
- Launch Monday or Tuesday for clean 7-day review alignment

### Week 3: 7-Day Review
- Pull performance data vs. account average
- Apply pause/keep/promote rules
- Update Change Log
- Move paused creative to ADV+ if applicable

### Week 4: Next Experiment Planning
- Review all active creative performance
- Identify top candidate for next experiment type
- Pull new messaging angles from value prop exercise if needed
- Begin client briefing for next creative assets

## Canva Workflow

### Access
- Login: Canva Pro account (Jim's access)
- Create a new folder for each client: `[Client Name] | Meta Ads`
- Inside the folder: create subfolders by experiment number

### Asset Creation Process

1. Start from the correct size template:
   - Feed square: 1080 × 1080px
   - Stories/Reels: 1080 × 1920px
   - Landscape: 1200 × 628px

2. Use Challenger's brand colours and fonts (from Client Workbook or brand guidelines)

3. For each image concept, create:
   - The core concept in 1:1
   - A 9:16 version if Stories/Reels are a priority placement for this client

4. Export as PNG (images) or MP4 (video) at highest quality

5. Name files clearly: `[Client]_[ExperimentN]_[AssetDescription].[ext]`
   Example: `Oaktree_Exp1_Lifestyle_CuttingBoard.png`

## Experiment Types and When to Use Each

### Ads Variety (Months 1–3)
**What:** Test multiple creative concepts with different visual approaches
**Best for:** Early experiments when no performance baseline exists
**Variables:** Images/videos (4-5 different concepts) + control copy
**Goal:** Find which creative format and visual direction resonates

### Copy Test (After Ads Variety Has Baseline)
**What:** Keep creative constant, test different headlines and primary texts
**Best for:** When you know which image/video wins but copy hasn't been isolated
**Variables:** 2-3 new headlines + 2-3 new primary texts, same winning image
**Goal:** Find the strongest copy combination for the winning creative

### Format Test (Any Point)
**What:** Static image vs. video vs. carousel with the same core message
**Best for:** When you have strong copy but don't know which format the audience prefers
**Variables:** Format type — keep copy and CTA the same across formats
**Goal:** Identify which creative format drives the lowest CPA

### Angle Test (Month 3+)
**What:** Test a completely new messaging angle vs. the current control
**Best for:** When the current control has been running for 60+ days and is showing fatigue
**Variables:** New messaging angle with matching image AND copy
**Goal:** Refresh the creative strategy with a new audience entry point

## Performance Benchmarks

Use these benchmarks to evaluate experiment results. These are category averages — Challenger's specific benchmarks should be established from their own account data after 30 days.

| Metric | Weak | Average | Strong |
|---|---|---|---|
| CTR (all) | Below 0.5% | 0.5–1.5% | Above 1.5% |
| CTR (link clicks) | Below 1% | 1–3% | Above 3% |
| CPM | Varies widely by audience and season | — | Lower = better delivery |
| Frequency | Above 3.5 = possible fatigue | 2–3 | — |
| Purchase CVR | Below 1% | 1–3% | Above 3% |

**Account-specific benchmark:** After 30 days, calculate the account average CPA from the main campaign. This becomes the benchmark — not industry averages. Experiments are evaluated as "better than account average," "at account average," or "worse than account average."

## Creative Fatigue Signals

A creative is fatiguing when:
- Frequency rises above 3.5 for the same audience
- CTR drops 20%+ compared to the same creative 2 weeks ago
- CPA rises 30%+ compared to the same creative's 30-day average

When fatigue is detected:
1. Check if it's the specific creative or the whole account (if all creatives are declining, it may be a seasonality/budget issue)
2. If confirmed fatigue: add new creative to the legacy set and reduce budget to the fatigued ad
3. Don't pause immediately — scale down first; let winning replacements absorb budget
