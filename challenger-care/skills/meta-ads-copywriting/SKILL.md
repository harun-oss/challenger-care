# Meta Ads Copywriting

## What this skill does

This skill generates ready-to-use Meta ad copy — primary texts, headlines, and video scripts — by first researching the brand from its website, then applying the right framework for the copy type requested.

All copy follows Challenger's proven format: structured Hook + Body + CTA, platform-correct character limits, and framework selection for headlines and video.

Use this skill when:
- Writing primary texts for a new DCT or flexible ad set
- Generating headline options for a creative test
- Briefing video creators with a structured script
- Refreshing stale copy with new angles

---

## Phase 1 — Establish Copy Type

Ask one question only:

> "What type of copy do you need?
>
> 1. **Primary Texts** — the main ad body (400–500 characters)
> 2. **Headlines** — short punchy lines (30–40 characters)
> 3. **Video Script** — voice + visual script with Hook, Body, and CTA
> 4. **All three** — full copy package for a new creative"

Wait for the answer before asking anything else.

---

## Phase 2 — Brand Research

Ask one question at a time — never two questions in the same message.

**Step 1:** Ask for the website URL.

**Step 2:** Once the URL is provided, research the brand thoroughly. Using web search or fetch, look for:
- All products or services offered
- Brand history and positioning
- Value proposition and key benefits
- Problems the brand/product solves
- Competitive advantages vs similar brands
- Tone of voice and brand personality

Summarise your findings in a brief internal brand profile before proceeding. Do not show this to the user — use it to inform the copy.

**Step 3:** Ask:
> "Do you want to focus on a particular product or service, or the business as a whole?"

Wait for the answer.

**Step 4:** Ask:
> "Which platform is this copy for? (e.g. Facebook Feed, Instagram Reels, Instagram Stories, Audience Network)"

Wait for the answer. Platform affects tone, length feel, and visual direction (especially for video scripts).

## Phase 3A — Primary Texts

*Use this phase only if the copy type selected is Primary Texts or All Three.*

**Rules:**
- Length: 400–500 characters per primary text (count carefully — do not go over 500 or under 400)
- Emojis: maximum 3 per primary text, placed naturally within the copy — not stacked at the end
- Quantity: generate 5 primary texts, each with a distinct hook angle

**Structure (each primary text must follow this layout with line breaks between sections):**

```
[Hook]

[Text Body]

[Call to Action]
```

**Hook:** A pattern-interrupt opening line that stops the scroll and speaks directly to the target audience's pain, desire, or curiosity. Do not start with the brand name.

**Text Body:** Explain what the product does and its key characteristics, then pivot to the main benefits for the customer. Keep it conversational — write like a person, not a brand.

**Call to Action:** One line that highlights the main offer and tells the reader exactly what to do next. (e.g. "Shop now and get free shipping on your first order.")

**Angle variety:** Each of the 5 primary texts should use a different hook angle — pain-point, aspiration, social proof, curiosity, and urgency. Do not repeat the same opening structure across texts.

Read `references/copy-frameworks.md` → Primary Text Angles for example angles and hooks by vertical.

## Phase 3B — Headlines

*Use this phase only if the copy type selected is Headlines or All Three.*

**Rules:**
- Length: 30–40 characters per headline (hard limit — 40 characters maximum)
- Quantity: generate 5 headlines
- No emojis in headlines — they rarely render correctly across placements

**Step 1:** Ask for the framework:

> "Which headline framework do you want to use?
>
> 1. Benefits Driven
> 2. Problem-Solution
> 3. Curiosity-Based
> 4. Fear of Missing Out (FOMO)
> 5. Testimonial-Style
> 6. How-To / Educational
> 7. Challenge-Based
> 8. Shock / Controversy
> 9. Social Proof / Popularity
> 10. Time-Sensitive / Urgency
> 11. Authority / Expert-Backed
> 12. Aspirational"

Wait for the answer.

**Step 2:** Generate 5 headlines using the chosen framework. Each headline should:
- Drive interest without feeling overtly promotional
- Tap into the audience's desires or needs
- Work as a standalone line (no context assumed from the primary text)

Read `references/copy-frameworks.md` → Headline Frameworks for examples and patterns by framework.

## Phase 3C — Video Script

*Use this phase only if the copy type selected is Video Script or All Three.*

**Rules:**
- Quantity: generate 5 video scripts
- Each script has 3 sections: Hook (first 3 seconds), Body (following ~30 seconds), CTA (last 3 seconds)
- Each section has two parts: **Voice Script** (what is said) and **Visual Script** (what is shown on screen)

**Step 1:** Before writing scripts, research: in addition to the brand website, search for consumer persona data for this type of product/service — who buys it, what motivates them, what they fear, what they aspire to.

**Step 2:** Ask for the framework:

> "Which video script framework do you want to use?
>
> 1. Problem-Agitate-Solution (PAS)
> 2. Hook-Story-Offer (HSO)
> 3. Attention-Interest-Desire-Action (AIDA)
> 4. Before-After-Bridge (BAB)
> 5. Problem-Solution-Outcome (PSO)
> 6. Gain-Logic-Fear (GLF)
> 7. Star-Story-Solution (SSS)
> 8. Features-Advantages-Benefits (FAB)
> 9. Open-Loop Curiosity (OLC)
> 10. Problem-Promise-Proof-Proposal (4Ps)
> 11. Challenge-Solution-Transformation (CST)
> 12. Fear-Doubt-Confidence (FDC)"

Wait for the answer.

**Step 3:** Generate 5 video scripts. Each must follow this structure:

```
Script [N] — [Framework] — [Angle name]

HOOK (First 3 seconds)
Voice: [Exact words spoken]
Visual: [What appears on screen — describe shot, text overlay, action]

BODY (Following ~30 seconds)
Voice: [Exact words spoken]
Visual: [What appears on screen — describe sequence of shots/visuals]

CTA (Last 3 seconds)
Voice: [Exact words spoken]
Visual: [What appears on screen — product shot, offer text, logo]
```

Each of the 5 scripts should use a distinct hook angle even within the same framework. Read `references/copy-frameworks.md` → Video Script Frameworks for pattern examples per framework.

If branding: read `../../../growthit-brand/assets/growthit-brand.md` for exact colours, fonts, and layout rules before building.

For plain text output: present each copy piece clearly labelled (Primary Text 1, Headline 3, Script 2, etc.) with visible section breaks.

## Error Handling

**Website cannot be fetched:** Ask the user to paste the brand's key product description, value proposition, and 2–3 competitor names directly into the chat. Proceed with the provided information.

**Character count violation:** Before presenting copy, count every headline and primary text character by character. If any piece violates the limit (headline >40 chars, primary text <400 or >500 chars), rewrite it silently before showing. Never present copy that breaks the character rules — this is a hard constraint.

**User doesn't want to choose a framework:** If the user says "you pick" for headlines or video framework, select the framework most appropriate for the brand's vertical and goals:
- eCommerce physical product → Benefits Driven (headlines) / Before-After-Bridge (video)
- SaaS / lead gen → Problem-Solution (headlines) / Problem-Agitate-Solution (video)
- Health/wellness → Aspirational (headlines) / Hook-Story-Offer (video)

**"All three" requested:** Complete Primary Texts first, then Headlines, then Video Scripts. Run brand research once and reuse it across all three — do not re-research between copy types.

## QA Gate

Before presenting copy:

- [ ] Character count verified — every headline is 30–40 chars, every primary text is 400–500 chars
- [ ] No more than 3 emojis in any single primary text
- [ ] Each primary text has distinct Hook, Body, and CTA separated by line breaks
- [ ] Each video script has Voice and Visual for all three sections (Hook / Body / CTA)
- [ ] 5 pieces of each copy type requested (not 3, not 4 — exactly 5)
- [ ] Hook angles are distinct across the 5 pieces — no repeated opening structure
- [ ] Framework selection confirmed before generating headlines or video scripts
