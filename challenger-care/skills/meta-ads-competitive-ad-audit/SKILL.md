# Meta Ads Competitive Ad Audit

**Owner:** the operator
**Timeline:** Week 2 — kickoff deliverable #5
**Outcome:** 8–12 competitor ads researched and analysed across creative format, hook, copy angle, and CTA. Competitive Ad Audit deck delivered to client with strategic implications for their own creative approach.

---

## Phase 1: Context & Competitor List

Ask the operator:
1. **Client name** — who is this audit for?
2. **Client's industry/vertical** — what market are they in?
3. **Competitor list** — which 2–4 competitors should we audit? (client may have mentioned them, or you can identify them from the Value Prop Exercise)
4. **Value Prop Exercise available?** — if completed, the competitor list and positioning gaps are already documented there

> Start with the competitors Challenger cares about most. If they haven't specified, use the top 2–3 brands that appear in their reviews ("I tried X before switching to them") or the ones competing for the same Meta Ads placements.

## Phase 2: Meta Ads Library Research

> Before researching, read `references/ads-library-research-guide.md` for search techniques, filter settings, and how to read the library data effectively.

**Access:** [Meta Ads Library](https://www.facebook.com/ads/library/?active_status=all&ad_type=all&country=US&is_targeted_country=false&media_type=all)

For each competitor (2–4 brands), pull 2–4 ads. Target: **8–12 ads total** across all competitors.

**Search approach:**
1. Search by brand name (exact match usually works best)
2. Filter: Country = US (or Challenger's primary market), All ad types, Active + Inactive
3. Filter by media type to see images vs. video separately
4. Look at ads that have been running for 30+ days — longevity signals performance

For each ad pulled, document:

```
Competitor: [Brand name]
Ad format: [Static image / Video / Carousel / Collection]
Duration running: [X days / weeks]
Hook (first line / first frame): [Exact text or description]
Primary text angle: [Problem-solution / Social proof / UGC / Testimonial / Offer / Other]
CTA button: [Shop Now / Learn More / Sign Up / Get Offer / Other]
Visual style: [Product-only / Lifestyle / UGC-style / Illustration / Text-heavy]
Key claim or differentiator used: [What do they lead with?]
```

## Phase 3: TikTok Ads Library Research

**Access:** [TikTok Ads Library](https://ads.tiktok.com/business/creativecenter/inspiration/topads/pc/en?region=US&period=30)

> How to download a video from the Meta Ads Library: _(Loom guide · ask the operator for the link)_

TikTok ads reveal creative angles that are increasingly crossing over into Meta. Even if Challenger isn't running on TikTok, competitor TikTok ads show what video hooks and formats are being tested in the market.

Search for the same 2–4 competitors. Pull 2–4 additional video ads.

Note for each TikTok ad:
- Hook (first 3 seconds — what happens?)
- Format (talking head, UGC, product demo, text-on-screen, voiceover)
- Whether the angle could be adapted for Meta video/Reels

## Phase 4: Analysis — Patterns & Gaps

After pulling all ads, analyse across the competitive set:

> For the full analysis framework including pattern recognition and gap identification, read `references/competitive-analysis-framework.md`.

### What Everyone Is Doing (Category Patterns)

Identify themes that appear across 3+ competitors — these are "table stakes" in the category. The client needs to acknowledge these but should not copy them.

Examples of category patterns to look for:
- "All competitors lead with price/discount offers"
- "All competitors use UGC/testimonial format video"
- "Everyone is running static product-on-white images"

### What No One Is Doing (Gaps / Opportunities)

Find angles, formats, or claims that none of the competitors are using. These are creative opportunities for Challenger:
- An emotional angle that's being underutilised
- A format (e.g., educational content, founder story) that no one runs
- A claim or differentiator that the competitors can't make but Challenger can

## Phase 7: Synthesis Brief

Before delivering the final audit, write a brief summary of key findings. This is used by the orchestrator to carry findings forward to the next skill in a multi-skill workflow.

**[Client Name] Competitive Creative Audit Key Findings**
- Top finding 1 — specific pattern across competitors. Example: "All 3 competitors lead with UGC-style video. Static creative is underutilized in the market"
- Top finding 2 — specific gap or opportunity. Example: "No competitor uses founder-led narrative. Opportunity to differentiate with authentic founder story"
- Top finding 3 — specific insight about hooks or angles. Example: "Pain-point hooks (frustration-focused) dominate competitors. Social proof angle is missing"

**Priority for downstream skills:** What the next skill should focus on. Example: "Next skill should test brand video and founder narrative to differentiate. Creative brief should emphasize the pain-point hook since competitors own it — consider owning social proof instead."

*If running standalone (not in an orchestrated chain), share this summary with the operator before the full deliverable.*
