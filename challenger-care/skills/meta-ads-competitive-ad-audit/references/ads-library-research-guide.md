# Meta Ads Library — Research Guide

## Accessing the Library

URL: https://www.facebook.com/ads/library/?active_status=all&ad_type=all&country=US

No login required. The library is publicly accessible.

## Search Techniques

### Finding Competitor Ads

**By brand name (most reliable):**
- Search the exact brand name — Meta matches to the advertiser account name
- If you get no results: try the brand's Facebook Page name (it may differ from the company name)
- For DTC brands: try both the brand name and the .com domain without "www"

**By keyword (for category research):**
- Search a category keyword (e.g., "collagen supplement", "project management software")
- Sort by "Most Recent" to see what's actively being tested
- Sort by date started (older = longer-running = likely proven performance)

**Common search issues:**
- If results seem thin: the brand may be running ads under a parent company or agency account name
- If results show a different brand: search more specifically or add the industry keyword

## Filter Settings for Competitive Research

| Filter | Recommended Setting | Why |
|---|---|---|
| Country | US (or client's primary market) | Focus on the relevant market |
| Ad type | All | Catch all formats |
| Active status | All (not just Active) | Inactive ads show what they tested and stopped |
| Media type | Rotate between All / Images / Videos | Analyse by format separately |
| Date range | Last 90 days | Captures current strategy + experiments |

## Reading the Library Results

### What Each Ad Shows You

**Duration running:** Visible as "Started running on [date]"
- 7–14 days: currently testing, no performance signal yet
- 30–60 days: running with some confidence, likely performing
- 90+ days: strong performer or evergreen creative — study carefully

**Number of ads:** How many active ads a brand runs indicates their testing volume
- 1–5 active ads: low testing volume, likely running with best performers
- 10–30 active ads: active testing, multiple DCT experiments likely
- 50+ active ads: high testing volume or many ad set variations — advanced advertiser

**Ad ID:** Each ad has a unique ID. Use this when documenting specific ads for the audit.

### What You Can't See

- Spend, impressions, or performance data (Meta does not expose this)
- Which audience the ad was shown to
- The campaign structure behind the ad
- Click-through rates or conversion data

**Workaround:** Longevity is a proxy for performance. An ad that has run for 60+ days without modification is almost certainly performing well.

## Creative Analysis Framework

When analysing each ad, go through these dimensions:

### Hook Analysis (First 3 Seconds / First Line)

The hook is the most important element. For video: what happens in the first 3 seconds? For static: what is the headline or dominant text overlay?

Hook types:
- **Question:** "Struggling with [pain]?" — works because people mentally answer
- **Bold claim:** "[Product] replaced my [expensive alternative]" — creates curiosity
- **Statistic:** "9 out of 10 [personas] experience [problem]" — establishes credibility
- **Relatability:** "[Common frustrating scenario]..." — pattern interrupt via recognition
- **Direct offer:** "50% off — today only" — transactional, attention via incentive

### Copy Angle Analysis

What is the primary persuasion strategy in the body copy?
- Problem-solution (pain → relief)
- Features → benefits
- Social proof (testimonial, reviews, user count)
- Urgency/scarcity (limited time, limited stock)
- Story/narrative (founder story, customer journey)
- Educational (teaches something relevant to the product)

### Visual Style Analysis

| Style | Description | Best For |
|---|---|---|
| Product-on-white | Clean product shot, no context | eCommerce, high-clarity products |
| Lifestyle | Product in use in aspirational setting | Aspirational brands, home, fashion |
| UGC-style | Shot like a phone video, casual | Trust-building, beauty, CPG |
| Text-heavy | Large text overlay, minimal image | Mobile-first, offer-led |
| Illustration | Drawn/graphic elements | SaaS, abstract products |
| Founder/people | Person speaking to camera | Brand trust, coaching, B2B |

### CTA Analysis

| CTA | Primary Use |
|---|---|
| Shop Now | eCommerce purchase intent |
| Learn More | Consideration/awareness |
| Sign Up | Lead gen, SaaS trials |
| Get Offer | Discount/promotion campaigns |
| Book Now | Service booking |
| Watch More | Video content/brand awareness |

## TikTok Ads Library Notes

URL: https://ads.tiktok.com/business/creativecenter/inspiration/topads/pc/en

The TikTok library shows top-performing ads by industry and region.

**Why research here even for Meta-only clients:**
- TikTok tests video hooks faster and at higher volume than most Meta advertisers
- Hooks that prove on TikTok often translate well to Meta Reels/Stories
- Can reveal emerging creative formats before they saturate Meta

**What to look for:**
- Video hook structure (text overlay, voiceover, or demonstration in first 3 seconds)
- Pacing (fast-cut vs. slower demonstration)
- Whether competitors are active on TikTok at all (signals their creative investment level)

**Downloading videos:**
- TikTok: use the download button visible on each ad
- Meta: use the Loom guide linked in the main SKILL.md or a browser extension — Meta doesn't have a native download button
